# NLP_tp_submission

This repository contains all the  TP files required for NLP class. 

Each file is named with its assigned TP.


The TP1_computational_Graph_exercices is pdf file. To access it, you can download the file. 

To run all the coding Labs (LAB 2,3,4,5,6,7) you need to download them and upload them in google colab.

For these specific labs : 

**LAB 5 - TEXT GENERATION WITH CLASSIFICATION MODELS**

When running the notebook, replace the cell token = "your_token_here" with the hugging face token you created

**LAB 6 - CLUSTERING AND MODELISATION**

Before running the notebook, you need to replace "GROQ_API_KEY" with the GROQ api key you created
